# GitHub MCP 테스트

이 파일은 GitHub MCP(Model Context Protocol) 기능을 테스트하기 위해 생성되었습니다.

## 테스트된 기능들

1. ✅ 사용자 정보 확인
2. ✅ 브랜치 목록 조회
3. ✅ 커밋 히스토리 조회
4. ✅ 코드 검색
5. ✅ 파일 내용 확인
6. ✅ Pull Request 목록 조회
7. ✅ 새 브랜치 생성
8. ✅ 파일 생성 및 푸시

- 생성일: 2025-10-30
- 생성자: ginishuh
